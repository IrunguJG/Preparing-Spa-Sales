Preparing Spa Sales – Excel Data Visualization Project

Project Overview:
This project focuses on transforming raw spa operational and sales data into meaningful insights using Microsoft Excel. The objective is to support management decision-making at Turquoise Oasis Spa through accurate, well-designed charts and visual analytics. 
The project demonstrates practical skills in Excel charting, data analysis, and dashboard-style reporting across sales, revenue, customer preferences, and operational progress.


Key Tasks & Skills Demonstrated: 
•	Data visualization using Pie, Line, Column, Bar, Area, Scatter, and Combo charts 
•	Use of PivotTables, data labels, chart styles, layouts, and sparklines 
•	Sales and revenue trend analysis 
•	Comparative analysis across massage types and time periods 
•	Formatting charts for professional presentation use 
•	Applying secondary axes, trendlines, and conditional formatting


Tools Used: 
•	Microsoft Excel 
•	Excel Charts & Sparklines 
•	Data Analysis & Reporting Techniques 


How to Use This Project: 
•	Download the Excel workbook from this repository. 
•	Open the file in Microsoft Excel (desktop version recommended). 
•	Navigate through the worksheets to view different charts and analyses. 
•	Review how raw data is transformed into visual insights for business decisions.


Learning Outcome: 
This project showcases my ability to clean, analyze, and visually present data using Excel in a real-world business scenario. It reflects strong analytical thinking, attention to detail, and professional reporting skills suitable for business and academic environments.
